stalwart

Helm chart for the Stalwart mail and collaboration server

Add repository

helm repo add kgrubb-stalwart https://kgrubb.github.io/stalwart-helm-chart
helm repo update

Install

helm install stalwart kgrubb-stalwart/stalwart \
  -n mail --create-namespace
